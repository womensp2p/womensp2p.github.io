Women's P2P toolkit webclient

This is a simple React Native application to handle the following functionalities

  • Ability to upload any media file (be it audio recording, text or video) or record from the application to send an alert post to all possible social media (Facebook, twitter, etc)
  • Offline queueing system to publish the messages when the internet bandwidth is available
  • Support to deploy the application to devices like IVR, Radios, brick phones which are being used frequently by the women from Kenya
  • Ability to send and receive messages

Women's P2P toolkit server

Lightweight server logic built in Django

  • Ability to provide APIs to receive messages(in any media) and upload to different social media websites(like facebook, twitter)
  • Explore possibilities for human and machine translation
  • Interaction with the account system to make appropriate DB calls to update record under every user
  • Integration logic to handle with devices like IVR, Radio and brick phones
  • Logic to record offline and upload at a scheduled time (better bandwidth in Kenya late at night)
